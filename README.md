# Türkiye CO2 Emissions and Industrial Development: Simple Linear Regression Analysis

     This project investigates the relationship between Türkiye’s industrial development and CO2 emissions between 1970 and 2023 using a simple linear regression model in R. The workflow includes importing annual data from Excel files, transforming country-level time-series observations into analysis-ready format, and building a regression model to assess the statistical association between industrial activity and CO2 emission levels.

     The repository also includes a visual representation of the model through a scatter plot and fitted regression line. As a portfolio project, it demonstrates core data analysis skills in R, including data cleaning, transformation, regression modeling, statistical interpretation, and basic visualization.
